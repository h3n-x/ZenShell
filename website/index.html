ZenShell Bot

Un bot de Discord todo-en-uno diseñado para transformar tu servidor en una comunidad vibrante y bien gestionada. Con funcionalidades avanzadas de moderación, un sistema de economía completo, reproducción de música, niveles personalizados y mucho más, ZenShell se adapta a las necesidades de cualquier servidor.

Características Principales

Moderación

Sistema completo de moderación con comandos para ban, kick, mute, warn y más. Gestión avanzada de roles y sistema anti-spam.

Economía

Sistema de monedas, tienda, comandos para ganar dinero y sistema de comandos personalizados que los usuarios pueden comprar.

Música

Reproduce música desde YouTube, controla la cola, ajusta el volumen y busca canciones fácilmente.

Niveles y Estadísticas

Sistema de experiencia, niveles, roles automáticos, logros desbloqueables y perfiles personalizados.

Eventos

Organiza sorteos, encuestas y configura recordatorios para mantener activa tu comunidad.

Base de Datos

Integración con Supabase para almacenar datos de usuarios, economía, logros y más de forma segura.

Comandos Principales

!ban @usuario [razón]

Banea a un usuario del servidor

!kick @usuario [razón]

Expulsa a un usuario del servidor

!mute @usuario [duración] [razón]

Silencia a un usuario por un tiempo determinado

!warn @usuario [razón]

Advierte a un usuario

!unban @usuario [razón]

Desbanea a un usuario y le envía una invitación

!balance [@usuario]

Muestra el balance de un usuario

!work

Trabaja para ganar monedas

!daily

Recibe monedas diarias

!shop

Muestra la tienda

!gamble <cantidad>

Apuesta monedas

!play <canción>

Reproduce una canción

!skip

Salta a la siguiente canción

!queue

Muestra la cola de reproducción

!volume <1-100>

Ajusta el volumen

!stop

Detiene la reproducción

!perfil [@usuario]

Muestra el perfil de un usuario

!logros [@usuario]

Muestra los logros de un usuario

!top [nivel/xp/monedas/logros]

Muestra el ranking

!role add @usuario @rol

Añade un rol a un usuario

!role remove @usuario @rol

Quita un rol a un usuario

!role create nombre color

Crea un nuevo rol

!role list

Lista todos los roles del servidor

!autoroles

Muestra los roles automáticos por nivel

Instalación y Configuración

1

Clonar Repositorio

git clone https://github.com/tu-usuario/Bot-Discord.git cd Bot-Discord
2

Instalar Dependencias

pip install -r requirements.txt
3

Configurar Variables de Entorno

Crea un archivo .env con las siguientes variables:

DISCORD_TOKEN=tu_token_de_discord URL_SUPABASE=tu_url_de_supabase SUPABASE_KEY=tu_clave_de_supabase
4

Configurar Base de Datos

Crea las tablas necesarias en Supabase (ver documentación)

5

Iniciar el Bot

python main.py

Preguntas Frecuentes

¿Cómo puedo añadir el bot a mi servidor?

+

Puedes añadir el bot a tu servidor haciendo clic en el botón "Añadir a Discord" en la parte superior de esta página. Necesitarás tener permisos de administrador en el servidor para poder añadirlo.

¿El bot es gratuito?

+

Sí, ZenShell es completamente gratuito y de código abierto. Puedes usar todas sus funciones sin costo alguno.

¿Cómo configuro la base de datos?

+

Necesitas crear una cuenta en Supabase y crear las tablas necesarias. En el README del repositorio encontrarás los scripts SQL para crear todas las tablas requeridas.

¿Puedo personalizar los comandos?

+

Sí, al ser de código abierto puedes modificar cualquier parte del bot para adaptarlo a tus necesidades. Además, el bot incluye un sistema de comandos personalizados que los usuarios pueden crear.

¿Cómo reporto un bug o sugiero una nueva función?

+

Puedes reportar bugs o sugerir nuevas funciones abriendo un issue en el repositorio de GitHub del proyecto.
